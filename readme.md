<!-- GETTING STARTED -->
## Getting Started

To get a local copy up and running follow these simple example steps.

### Installation

* requirements
  ```sh
  pip install -r requirements.txt
  ```
* streamlit app
* ```sh
  streamlit run TEST_PAGE.py
  ```
